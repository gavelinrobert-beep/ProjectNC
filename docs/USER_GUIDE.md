## Dashboard

View key metrics and quick access to critical functions.

### Key Metrics
- Total Assets: Fleet size
- Mobile Assets: Currently active
- Active Tasks: Ongoing work
- Critical Alerts: Issues requiring attention
- Low Fuel: Assets needing refueling
- Maintenance Due: Service required

---

## Live Map

Real-time tracking of all assets and locations.

### Controls
- Zoom: `+/-` or mouse wheel
- Pan: Click and drag
- Layer toggles: Show/hide assets, bases, zones
- Click markers for details

### Asset Information
Click any asset to view:
- Current status
- Fuel and battery levels
- Speed and location
- Assigned mission

---

## Fleet & Resources

Manage vehicles and equipment.

### Adding Assets
*Requires: Admin role*
1. Click **+ Add Asset**
2. Fill in details (name, type, fuel type, capacity)
3. Click **Create**

### Updating Status
*Field Operators*
1. Select asset
2. Update status, fuel level, notes
3. Click **Save**

---

## Tasks & Missions

Plan and track work assignments.

### Creating Missions
1. Click **+ New Mission**
2. Set name, type, priority
3. Add waypoints on map (minimum 2)
4. Review estimates
5. Click **Create**

### Mission Types
- **Patrol**: Regular inspection routes
- **Transfer**: Move resources between bases
- **Supply**: Deliver materials
- **Emergency**: Priority response

---

## Inventory Management

Track supplies and materials.

### Adding Items
1. Click **+ Add Item**
2. Enter name, quantity, location
3. Set minimum stock level
4. Click **Save**

### Stock Transactions
- Consume: Used in field
- Transfer: Moved to location
- Restock: Added to inventory

---

## Field Reports

Submit status updates from the field.

### Creating Reports
1. Click **+ New Report**
2. Add title, type, description
3. Attach photos (optional)
4. Submit

Reports work offline and sync when connected.

---

## Administration

*Municipality Admin role only*

### User Management
- Add/remove users
- Assign roles
- Manage permissions

### Roles
- **Municipality Admin**: Full access
- **Contractor**: Limited admin
- **Field Operator**: Update assets
- **Viewer**: Read-only

### Reports & Export
Generate reports for:
- Asset usage
- Fuel consumption
- Maintenance history
- Inventory transactions

Export formats: PDF, CSV
